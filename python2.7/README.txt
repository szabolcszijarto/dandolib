
What you need to make this library work in python2.7:
(or at least what I used... porting to other versions might be possible but I haven't tried)

1. Ubuntu linux
2. python 2.7 (I used 2.7.6)
3. pygattlib , a python library for the GATT protocol used by Bluetooth LE devices.
   Available here: https://bitbucket.org/OscarAcena/pygattlib
